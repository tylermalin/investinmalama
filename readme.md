Interactive investor dashboard for Seed Series A fundraising.

## Quick Start

1. **Open locally:** Open `index.html` in your browser.
2. **Deploy to Vercel:** Push to GitHub and connect to Vercel.
3. **Customize:** Edit investor name and metrics directly in `index.html`.

## Features

- **SAFE Investment Calculator**: Model ownership based on investment amount and valuation caps.
- **Printable Term Sheet**: Generate one-click professional summaries for investors.
- **Investor Data Room**: Secure access to legal terms and strategic scaling plans.
- **dMRV Transparency**: Real-time project tracking and verification metrics.
- **Project Dynamics**: Integrated view of Hawaii and Idaho project economics.

## Deployment

### Vercel (Recommended)

1. Push to GitHub.
2. Visit [Vercel](https://vercel.com).
3. Import this repository.
4. Click "Deploy".
5. Your portal will be live at `https://investinmalama.vercel.app`.

### GitHub Workflow

```bash
git add .
git commit -m "update: rename to index.html for vercel"
git push origin main
```
